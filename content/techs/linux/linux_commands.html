ALIAS

Descrição
Permite criar atalhos (apelidos) para comandos no Shell do Linux

Sintaxe:
alias apelido=comando

Exemplos:
alias l="ls -lga"
alias kg="kubectl get"

Compatibilidade: Debian & RHEL based distros

Importante: O alias criado diretamente no prompt do shell só é válido para a sessão corrente.
Para torná-lo permanente é preciso incluí-lo em um script de carregamento (.bashrc, por exemplo)

APT-CACHE

Descrição
Permite a instalação e gerenciamento de pacotes de software em distribuições baseadas no Debian.

Sintaxe:
apt-get install nome-do-pacote
apt-get update/upgrade/purge/etc...

Exemplos:
alias l="ls -lga"
alias kg="kubectl get"

Compatibilidade: Debian & RHEL based distros

Comando Descrição Exemplo Observação
pkgnames Mostra a lista de todos os pacotes disponíveis apt-cache pkgnames Para filtrar use: apt-cache pkgnames nome_pacote
search Mostra o nome e a descrição do pacote apt-cache search nome_pacote
show Mostra todas as infomações do pacote apt-cache show nome_pacote
showpkg Mostra as dependencias de um pacote específico apt-cache showpkg nome_pacote
stats Mostra estatisticas do cache apt-cache stats

APT-GET

Descrição
Permite a instalação e gerenciamento de pacotes de software em distribuições baseadas no Debian.

Sintaxe:
apt-get install nome-do-pacote
apt-get update/upgrade/purge/etc...

Exemplos:
alias l="ls -lga"
alias kg="kubectl get"

Compatibilidade: Debian & RHEL based distros


Esta opção vai além do autoclean pois limpa também os arquivos .DEB. Para ver o espaço utilizado pelos pacotes .DEB use: sudo du -sh /var/cache/apt/archives
Comando Descrição Exemplo Observação
update Atualiza a lista de pacotes disponíveis apt-get update ou apt update
upgrade Atualiza as versões dos pacotes disponiveis apt-get upgrade ou apt upgrade Atualiza o que tem uma nova versão disponível para a plataforma, como definido em /etc/apt/sources.list ou em /etc/apt/sources.list.d/.
Install Instala o sofware referente ao pacote solicitado apt-get install nginx ou apt install nome_pacote
remove Remove pacotes da lista de disponíveis apt-get remove nginx ou apt remove nome_pacote Deixa os arquivos de configuração no sistema
autoremove Remove os pacotes que não estão em uso da lista de disponíveis apt-get autoremove ou apt autoremove Deixa os arquivos de configuração no sistema
purge Remove o pacote solicitado da lista de disponíveis e seus arquivo de configuração apt-get purge nginx ou apt purge nome_pacote Remove também os arquivos de configuração dos pacotes
Também pode ser usado assim: apt-get remove --purge nome_pacote
source Baixa o codigo fonte do pacote solicitado apt-get source nome_pacote O pacote não aparecerá na lista de disponíveis
É comum ocorrer o erro: You must put some ‘source’ URIs in your sources.list
Neste caso é preciso localizar a URL para o arquivo "deb-src" do pacote e
incluir ela no arquivo /etc/apt/sources.list
Link para troubleshooting
Exemplo (NGINX)
Para fazer download, desempacotamento e build do pacote use: sudo apt-get --compile source goaccess
build-dep Instala as dependencias de build do repositorio solicitado apt-get build-dep nome_pacote Resolve dependencias de "build-time", as de "runtime" são resolvidas pelo comando "Install"
dist-upgrade Atualiza a lista de pacotes de forma integigente, removendo e/ou instalando o que for necessário pra atualizar apt-get dist-upgrade apt-get upgrade não remove pacotes, já o dist-upgrade remove, se for necessário para atualizar
dselect-upgrade remove os pacotes que não estão na lista e instala tudo que estiver nela apt-get dselect-upgrade É mto util pra instalar pacotes que estão em uma máquina em outra.
É necessário fazer o 'backup' da lista da máquina original antes:
dpkg --get-selection > packages.list
E aplicá-lo na nova:
apt-get dselect-upgrade
autoclean Remove os arquivos de pacotes (.deb) que não podem mais ser baixados (versões antigas…) e são completamente inúteis e obsoletos. apt-get autoclean
clean apt-get clean Limpa os arquivos baixados para instalação dos pacotes Ótimo para economizar espaço eliminando arquivos de instalação que não estão mais sendo utilizados
Interessante utilizar em Dockerfiles pra diminuir o tamanho da imagem
check Verifica se não há dependencias quebradas e atualiza o cache de pacotes apt-get check Ferramenta para diagnóstico da integridade dos pacotes instalados
changelog Mostra o log de alterações e a versão atual do pacote selecionado apt-get changelog nome_pacote
download Faz o download de um pacote sem instalá-lo apt-get download nome_pacote
